ffftp is a minimal ftp client for casual ftp users.

Download for windows
Mac build coming soon

Features

  • Save connections to favorites for easy-access
  • Drag from file explorer to upload
  • Simple file renaming, deleting, and downloading
  • Doesn't look like it came straight from Windows XP
  • Built on Electron/Angular
  • Open source
